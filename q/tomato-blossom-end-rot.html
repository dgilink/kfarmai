방울토마토 밑부분 검게 썩음 | 배꼽썩음병과 칼슘장해

토마토 열매 밑부분이 검게 마르거나 썩어 보이면 배꼽썩음 증상일 가능성이 큽니다. 병원균보다 칼슘 이동 불량, 물관리 불균형, 고온·건조, 뿌리장해와 관련된 생리장해인 경우가 많습니다. 열매 끝이 검게 함몰되고 마른 가죽처럼 변하면 병해보다 생리장해 가능성을 먼저 확인합니다.

가능한 원인

  • 칼슘 흡수·이동 불량
  • 건조와 과습이 반복되는 물관리 불균형
  • 고온과 빠른 생장
  • 뿌리 손상 또는 염류장해
  • 질소 과다로 생육이 너무 왕성한 경우

사진으로 확인할 포인트

  • 검은 부분이 열매 끝, 즉 배꼽 부위인지 확인
  • 곰팡이성 부패 냄새가 있는지 확인
  • 최근 물주기 간격과 토양 건조 정도 확인
  • 잎과 줄기 생육이 과도한지 확인
  • 여러 열매에서 반복되는지 확인

바로 해야 할 조치

  • 물주기를 일정하게 유지합니다.
  • 뿌리 환경과 배수 상태를 점검합니다.
  • 과도한 질소 시비를 줄입니다.
  • 칼슘제 사용은 농도와 사용법을 지켜 보조적으로 활용합니다.
